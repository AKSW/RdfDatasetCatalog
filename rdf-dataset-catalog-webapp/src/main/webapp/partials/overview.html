What is it?

This is a simple minimal registry for RDF datasets, where you can specify respective download and Sparql endpoint locations.

What is it NOT?

This is not a generic dataset repository where you can add arbitrary data and describe them in detail.

Why?

To provide a high quality repository for RDF datasets together with references to where to get the actual data and respective meta data. Clients can then discover the datasets and their relations by querying the Sparql Endpoint.

How?

Want to share an RDF dataset hosted as a download and/or in a SPARQL endpoint? Register it here! If applicable, add relations to other datasets, such as 'isVoidDescriptionOf'.
